# F12 menu:  

- Change the saturation and brightness of the sky.
- Change the saturation and brightness of the stars.
- Random sky texture for every raid
- Time of day visibility multiplier (make the stars appear sooner or later in the day)

# Custom skies tutorial:

1. Download Unity 2022.3.43f1.
2. Create a new project.
3. Drag and drop the contents of the UnityAssetsSource folder from the mod repository.
4. Add your own nightsky textures (2:1 aspect ratio) by dragging them into the asset window.
5. In the inspector, disable mipmap generation, filter mode to trilinear, Max Size to whatever you want, compression to High Quality, at the bottom set AssetBundle to nightsky.bundle
6. Go to Tools/Night Sky and click Build bundle.
7. Place the created bundle in your mod folder and even create an addon if you want to.
8. Enjoy !
